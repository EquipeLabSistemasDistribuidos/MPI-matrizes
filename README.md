# MPI matrizes
## Dependências no Linux (ou WSL no Windows) do RMI

```
sudo apt install openssh-client
```
```
sudo apt install openssh-server
```

```
sudo apt install libopenmpi-dev
```

```
sudo apt install libopenmpi3
```
